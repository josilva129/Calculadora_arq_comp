# 🔢 SPTecher's Machine

Conversor de bases numéricas desenvolvido em HTML, CSS e JavaScript.

O sistema permite converter números entre as seguintes bases:

- Decimal
- Binário
- Octal
- Hexadecimal

## 🚀 Funcionalidades

✔ Converter Decimal → Binário, Octal e Hexadecimal  
✔ Converter Binário → Decimal, Octal e Hexadecimal  
✔ Converter Octal → Decimal, Binário e Hexadecimal  
✔ Converter Hexadecimal → Decimal, Binário e Octal
